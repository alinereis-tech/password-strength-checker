# 🔐 Password Strength Checker

A simple Python program that evaluates the strength of a password based on security criteria.

## 📌 Features

- Checks password length
- Detects lowercase and uppercase letters
- Identifies numbers
- Detects special characters
- Classifies passwords as:
  - Weak
  - Medium
  - Strong

## 🚀 How it works

The program assigns a score based on different characteristics of the password:

- Length (at least 8 characters)
- Contains lowercase letters
- Contains uppercase letters
- Contains numbers
- Contains special characters

Based on the score, the password is classified into one of three categories.
